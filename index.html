# WelCome Friend

起初我们用抽象的表达方式去理解这个世界万物，但时至今日我们追求的东西都是抽象的。

# 为什么做这么一个空间?

鄙人不才，就这么一点兴趣爱好。想要将所见所闻，所想所感都写在这里面，当然不是胡乱编撰和自我感动。就好像有时候女朋友和我煲电话粥，我都能对着电话那头刚洗漱完的她分享容器、微服务、容器编排、云原生等一些列专业名词。即使我知道她听不懂，不过她还是很认真的听我诉说那些对于他来说是火星语的东西，这也是为什么我那么爱她的原因之一。

# 内容

我的初步构想，这里会首先记载在技术开发学习中遇到的技术积累笔记。通俗点说叫它技术心得会更符合我的语风，可能对于我自己起到了一个快速查阅的好处，但更希望能通过我的示例帮助到正在阅读这句话的你,要知道我爱你。

# 想对你说

我相信是缘分让我们相遇，通过你那一头的光纤和我这一头的光纤，途径若干路由与网关让我们认识彼此，想想网络的发明真是一件伟大的事。

# 24 小时很公平

这世上我认为绝对公平的事情就是 24 小时制，无论男女老少都无法横跨这个维度的制约。虽然时间是抽象的，看不见的。但他就是能制约这个自然界的万物法则。
